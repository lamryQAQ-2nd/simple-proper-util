Testing proper-huk.js

This page allows you to test the functionalities of the proper-huk.js utility. Follow the instructions in each section to check each function's output.

Test testProperExists function

Enter a JSON object and a property name. Click "Test" to check if the property exists in the given object.

Test getProper function

Enter a path string to retrieve the value at that path from the given object.

Test putProper function

Enter a path and a value to set the value at the specified path in the given object. The updated object will be displayed.
